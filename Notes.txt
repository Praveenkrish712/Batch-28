HTML5 - Hyper text markup Language (<head> content <head/>)

Self Closing - Hr, Meta, Br, img, input

* It is used to create a Web page
* It is not a Programming Language
* It is a markup Language
* EWvery Website Can start with HTML

1) Heading - h1 to h6

2) Paragraph - p

3) Hr

4) Break tag - br 

5) Text Formatting - b or strong, u, i, sub, sup and mark

6) Anchor Tag (linking) - a

7) Image Tag - img

8) List - Ordered list and Unordered list

9) Tables - tr, th and td

10) Forms - Label, input, placeholder, required, button, Type(important), fieldset, legend

11) Media Elemets - Video and Audio

12) Semantic Tags - Header, footer, section, nav and Main

13) Division Tag - Div

14) Span
