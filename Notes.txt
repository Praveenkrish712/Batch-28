CSS - Cascading style sheet
====> It is used to style and design a webpage
====> CSS works together with HTML
====> Change the Text,color,background
====> To set the image and design a layout
====> Adding Animation and Transition

1) Types of CSS
i) Inline CSS 
ii) Internal CSS
iii) External CSS
Inline > Internal > External

2) CSS Syntax
Selector{
    property: value;
}

3) Types of Selector
i) Elemets Selector - (h1) (h2) (p)
ii) Class Selector - (.) - Mulitiple - we can use the same class name no of times
iii) Id Selector - (#) - Unique - only one time we can use the same id name
iv) Universal Selector - (*) - Give some style it will apply to the entire site
v) Group Selector - Class or id Selector contains different tags, in that different tags we want to change the same style means we can go with Group Selector

6) Box Modeling 
==> content
==> Padding - Around the content is called padding
==> Border - Around the padding is called Border
==> Margin - Around the border is called Margin        

7) Position
i) Static (default)
ii) Relative - Moving one place to another place (set the limit like left,right,top and bottom)
iii) Absolute - Absolute can search the non-Static parent, once reach the non-Static parent it will act as a same relative, otherwise it will search one more parent..... there is no non-Static parent means it will directly go to the root(body) of the website/webpage
iv) Fixed - to be fixed to the place
v) Sticky - using top:0 once reach the top:0 it will act as a same fixed

8) Display - Block(default), Inline, Inline-Block, flex, grid and None

         Flex                               Grid
   One Dimensional layout           Two Dimensional layout
   Row OR Column                       Row and Column
   Best for small Componenets       Best for Full page layout
   Ex: Navbar, cards                 Ex: Section, Dashboard

======> Flex ---------------
i) Using Display:flex; (Column into row)   
ii) Flex direction - Row, row-reverse, Column, column-reverse
iii) Justify-content (X) - Start, end, center, space-Around, space-evenly, space-between
iv) Align-items (Y) - (Set the height) - Start, center, end
v) Gap - to be set the values (px,%,rem,cm,m,vh,vw)
vi) Flex-wrap - nowrap, wrap, wrap-reverse
vii) Align-content (Y) - (Set the height) - (Enable the wrap) - Start, end, center, space-Around, space-evenly, space-between
viii) Flex on items - Flex: 1, 2, 3
ix) Order - Change the visual order of the property

=======> Grid ---------------
i) Using Display:grid; (No change)
ii) grid-template-columns: 300px 400px 500px;
                         : 30% 40% 30%;
                         : 1fr 1fr 1fr;
                         : 200px 30% 1fr;
                         : repeat(4,300px);
                         : minmax(200px,1fr) 1fr 1fr;
                         : repeat(3,minmax(200px,1fr)); 
iii) grid-template-rows: 300px 400px 500px;
                       : 30% 40% 30%;
                         : 1fr 1fr 1fr;
                         : 200px 30% 1fr;
                         : repeat(4,300px);     
iv) grid-column-start: 1; (starting value)
    grid-column-end: 4; (Ending value)   
    grid-column: span 3;
v)    grid-row-start: 1; (starting value)
      grid-row-end: 4; (Ending value)   
      grid-row: span 3; 
vi) grid-template-areas: 
    "h h h h"
    "sb1 m m sb2"
